Employee Attrition Major Project - Complete Version
-----------------------------------------------
How to run:
1. Extract this folder and open in VS Code
2. Create virtual env and activate: python -m venv venv  then venv\Scripts\activate
3. Install requirements: pip install -r requirements.txt
4. Run app: python app.py
5. Open http://127.0.0.1:5000 in browser. Dashboard: http://127.0.0.1:5000/dashboard
